API Documentation

Base URL: localhost:3000/api

Show Routes

  • /show/add/inception - Create a show
  • /show/update/inception - Update a show
  • /show/delete/inception - Delete a show
  • /show/allshows - Get all shows
  • /show/inception - Get a show by ID

User Routes

  • /user/signup/777 - Create a user
  • /user/allusers - Get all users
  • /user/777 - Get a user by ID
  • /user/delete/777 - Delete a user
  • /user/update/777 - Update a user
  • /user/777/addtolist/favorites - Add a show to a user's list
  • /user/777/removefromlist/favorites - Remove a show from a user's list
  • /user/777/addfriend/:friendFirebaseId - Add a friend to a user's list
  • /user/777/removefriend/:friendFirebaseId - Remove a friend from a user's list
  • /user/777/addcomment - Add a comment
  • /user/777/deletecomment - Delete a comment
